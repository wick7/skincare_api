# Skincare API

Express.js API with GET, POST, and PUT routes. For a small MySQL db containing data on purchased skin care products.

## Database

\*Contact owner for DB schema

## Installation

```bash
npm i
```

## Usage

Create .env file

```
DB_HOST=""
DB_PORT=
DB_DATABASE=""
DB_USERNAME=""
DB_PASSWORD=""
```

Run

```bash
npm run dev
```
